# CRM WhatsApp -- PRD Overview
Versao 1.0 | Abril 2026

## Objetivo
Plataforma CRM com funil Kanban + espelhamento WhatsApp via Evolution API.
4+ instancias simultaneas, tempo real, audio bidirecional.

## Modulos
1. Auth + RBAC (4 roles)
2. Leads + Kanban Drag-and-Drop
3. Chat WhatsApp espelhado
4. Gestao de Instancias
5. Dashboard + Relatorios

## Stack
Ver PROMPT-MASTER-CRM.md para stack completo.

## Criterio de sucesso
- Lead criado automaticamente via WhatsApp em < 3 segundos
- Kanban atualizado em tempo real para todos os operadores
- 4 instancias WhatsApp estaveis por 72h
